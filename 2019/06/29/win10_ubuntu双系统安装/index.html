Win10和Ubuntu双系统安装记录

Posted on 2019-06-29

换了新电脑，可以把旧电脑安上双系统玩了。

因为没截图，所以到网上去找了一些图，全当留个教程。

首先把资料备份，腾出来一个盘作为Ubuntu的系统盘，这里分配了大概50G，然后把放电影的那个盘扩容到了300G，大概是整个硬盘的一多半了。用到的软件是diskgenius，很强大。

然后我在机械硬盘里的e盘划分了空闲磁盘，步骤如下：
按住win+X键，在菜单里找打disk management，进入。

选那个e盘，然后右击选择shrink volume。压缩后已经有了50个G是未分配状态了，新安装的Ubuntu就可以放在这里。

禁用快速启动需要说明一下了，网上很多教程都说需要关闭快速启动选项，不过我没有关闭，并没有什么影响，建议还是关了吧，毕竟多一事不如少一事。

制作U盘启动盘用到的是UltralSO，选择下载的iso镜像，打开。选择“启动”中的写入磁盘映像，在“硬盘驱动器”选择你的U盘，这会格式化的U盘，确保你的U盘没有数据，或者数据不重要，其他默认，点击写入即可。


接下来就是进入BIOS，联想的一般是狂按F2，“Secure Boot”设置成Disabled，然后把一个Option设成UEFI的。这里查了一下，Secure Boot设计之初作用是防止恶意软件侵入。事实上它能够做到的仅仅是，当电脑引导器被病毒修改之后，它会给出提醒并拒绝启动，避免可能带来的进一步损失。

然后插上U盘，启动后进入了Ubuntu的界面，在“准备安装”菜单，可以不勾选“为图形和无线硬件，以及 MP3 和媒体安装第三方软件”，这样子安装过程能快些，而这些东西以后也能通过系统更新来安装。然后进入“安装类型”菜单，这里必须选择“其他选项”。

接下来就是分区，这里很坑，有一个“/”和“/boot”搞错了，网上的教程不对，这里根据我自己电脑的配置进行分区。

点击“安装ubuntu”，进行“自定义安装”，开始配置分区：

  • 点击空闲空间，大小：至少15G；用于：Ext4日志文件系统；挂载点：/ ;
  • 点击空闲空间，大小：至少200M；用于：Ext4日志文件系统；挂载点：/boot ;
  • 点击空闲空间，大小：至少4G；挂载点：交换空间 ;
  • 点击空闲空间，大小：剩余空间；用于：Ext4日志文件系统；挂载点：/home ;
  • 安装启动引导器的设备选择“/boot”所在分区；

等待安装就行了。

安装成功后重启应该能进到windows里去，最后使用easyBCD修改一下引导，一张图说明步骤：

最后每次开机的时候就有一个选项，可以选择进入哪个系统。
